Web Application Security Lab

A hands-on exploration of how web applications are built and how they can be secured or exploited.

What is this?

This is a deliberately simple web application built to understand how HTML, CSS, JavaScript, and backend code work together — and where security vulnerabilities can be introduced at each layer.

Layers of a web application

  • Front-end: HTML structure, CSS presentation, JavaScript behaviour
  • Back-end: Server-side logic, database queries, API endpoints
  • Database: Persistent storage — where sensitive data lives
